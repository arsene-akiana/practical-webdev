# practical-webdev

Web development exercises and projects using **HTML, CSS, JavaScript, and PHP** — completed as part of my Software Engineering studies at **CFI-CIRAS** and Full Stack training at **Akieni Academy** (Brazzaville, Congo).

This repository covers the full web stack, from semantic markup to server-side scripting.

---

## 📁 Repository Structure

```
practical-webdev/
│
├── html/               # Semantic structure, forms, accessibility
├── css/                # Layouts, Flexbox, Grid, animations
├── javascript/         # DOM, events, ES6+, async programming
├── php/                # Server-side logic, forms, file handling
├── projects/           # Full mini-websites combining all layers
└── ...                 # More to come
```

---

## 🧠 Concepts Covered

- **HTML5** — semantic elements, forms, multimedia, accessibility
- **CSS3** — Flexbox, Grid, responsive design, animations
- **JavaScript** — DOM manipulation, events, ES6+, fetch API, async/await
- **PHP** — variables, functions, OOP, form handling, server-side logic
- *And more as the learning progresses...*

---

## 🎯 Why Web Development?

Web development is the most direct path to building products people use every day. Mastering both the client and server sides means being able to take an idea from design to deployment, independently.
